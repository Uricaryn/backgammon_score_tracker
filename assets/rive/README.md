# Rive assets (deprecated)

This app no longer uses `.riv` files. Board effects use pub packages instead:

- **Dice faces:** [`roll_dice_2d`](https://pub.dev/packages/roll_dice_2d) (bundled PNGs)
- **Win celebration:** [`confetti`](https://pub.dev/packages/confetti) (code-only particles)
- **Hit burst:** CustomPaint in `board_effects_overlay.dart`

See [docs/art_pipeline.md](../../docs/art_pipeline.md).
